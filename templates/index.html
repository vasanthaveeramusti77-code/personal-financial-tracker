Personal Financial Tracker

This personal financial tracker website is designed to help users manage their income and expenses efficiently. It includes a secure signup and login system using Flask and SQLite, ensuring each user's data is private and accessible only after authentication. Once logged in, users can add, edit, and delete their financial transactions on a personalized dashboard. The interface is simple, responsive, and user-friendly, allowing for easy tracking of daily, weekly, or monthly budgets. Built using HTML, CSS, JavaScript, Python, and SQL, this project demonstrates full-stack web development from frontend design to backend logic and database integration.
